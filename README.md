# Login Nexus Blazor WASM

Projeto desenvolvido para a disciplina de Interação Humano Computador e UX.

## Descrição

O sistema simula uma plataforma acadêmica com autenticação de usuários utilizando Blazor WebAssembly.

O projeto possui:

- Tela de Login
- Autenticação mockada
- Dashboard do usuário
- Redirecionamento de páginas
- Serviço de autenticação com Dependency Injection

## Tecnologias Utilizadas

- C#
- Blazor WebAssembly
- ASP.NET Core
- .NET 8
- Razor Components

## Estrutura do Projeto

- `Pages` → páginas da aplicação
- `Models` → modelos de dados
- `Services` → serviços de autenticação
- `Layout` → menu e layout principal
